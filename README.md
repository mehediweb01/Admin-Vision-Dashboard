# Admin Vision Dashboard
A modern admin dashboard built for managing analytics, users, and UI components efficiently using a clean and responsive UI.

### 📝 Project overview: 
- Designed and developed a modern admin dashboard interface focused on clean UI/UX, responsive layouts, and reusable components. The project showcases a complete dashboard design system with multiple administrative pages, analytics sections, data tables, charts, and form layouts without backend integration.


## Installation️ & Setup
1. Clone the repo and install dependencies:

```bash
git clone https://github.com/mehediweb01/Admin-Vision-Dashboard.git
cd admin-vision-dashboard
npm install
```

2. Run the application:

```bash
npm run dev
```
